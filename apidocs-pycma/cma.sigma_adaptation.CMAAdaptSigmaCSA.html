cma.sigma_adaptation.CMAAdaptSigmaCSA(CMAAdaptSigmaBase) class documentation

Part of cma.sigma_adaptation (View In Hierarchy)

CSA cumulative step-size adaptation AKA path length control.

As of 2017, CSA is considered as the default step-size control method within CMA-ES.

overrides cma.sigma_adaptation.CMAAdaptSigmaBase._update_ps

update path with isotropic delta mean, possibly clipped.

From input argument es, the attributes isotropic_mean_shift, opts['CSA_clip_length_value'], and N are used. opts['CSA_clip_length_value'] can be a single value, the upper bound parameter, such that:

max_len = sqrt(N) + opts['CSA_clip_length_value'] * N / (N+2)

or a list with lower and upper bound parameters.
