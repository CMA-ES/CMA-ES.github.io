collection of many meta parameters.

Meta parameters are either annotated constants or refer to options from CMAOptions or are arguments to fmin or to the NoiseHandler class constructor.

MetaParameters take only effect if the source code is modified by a meta parameter weaver module searching for ## meta_parameters.... and modifying the next line.

CAVEAT
