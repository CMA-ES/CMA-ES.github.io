class MetaParameters(object):

View In Hierarchy

collection of many meta parameters.

Meta parameters are either annotated constants or refer to options from CMAOptions or are arguments to fmin or to the NoiseHandler class constructor.

MetaParameters take only effect if the source code is modified by a meta parameter weaver module searching for ## meta_parameters.... and modifying the next line.

Details

This code contains a single class instance meta_parameters

Some interfaces rely on parameters being either int or float only. More sophisticated choices are implemented via choice_value = {1: 'this', 2: 'or that'}[int_param_value] here.

CAVEAT

meta_parameters should not be used to determine default arguments, because these are assigned only once and for all during module import.
