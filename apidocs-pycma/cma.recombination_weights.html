cma.recombination_weights module documentation

Part of cma

Define a list of recombination weights for the CMA-ES. The most delicate part is the correct setting of negative weights depending on learning rates to prevent negative definite matrices when using the weights in the covariance matrix update.

The dependency chain is

lambda -> weights -> mueff -> c1, cmu -> negative weights

Class RecombinationWeights a list of decreasing (recombination) weight values.
