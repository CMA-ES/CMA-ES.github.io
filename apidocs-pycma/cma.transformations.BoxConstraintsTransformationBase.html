cma.transformations.BoxConstraintsTransformationBase(object) class documentation

Part of cma.transformations (View In Hierarchy)

Known subclasses: cma.transformations._BoxConstraintsTransformationTemplate, cma.transformations.BoxConstraintsLinQuadTransformation

Implements a transformation into boundaries and is used in top level boundary handling classes.

Example:

tf = BoxConstraintsTransformationAnyDerivedClass([[1, 4]])
x = [3, 2, 4.4]
y = tf(x)  # "repaired" solution
print(tf([2.5]))  # middle value is never changed
[2.5]
assert all([yi <= 4 for yi in y])
