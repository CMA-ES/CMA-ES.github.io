cma API Documentation

Module Index

  • cma - Package cma implements the CMA-ES (Covariance Matrix Adaptation Evolution Strategy).
    • cma.bbobbenchmarks - BBOB noiseless testbed.
    • cma.constraints_handler - A collection of boundary and (in future) constraints handling classes.
    • cma.evolution_strategy - CMA-ES (evolution strategy), the main sub-module of cma providing in particular CMAOptions, CMAEvolutionStrategy, and fmin
    • cma.fitness_functions - versatile container for test objective functions.
    • cma.fitness_models - Fitness surrogate model classes and handler for incremental evaluations.
    • cma.fitness_transformations - Wrapper for objective functions like noise, rotation, gluing args
    • cma.interfaces - Very few interface defining base class definitions
    • cma.logger - logger class mainly to be used with CMAEvolutionStrategy
    • cma.optimization_tools - Utility classes and functionalities loosely related to optimization
    • cma.purecma - A minimalistic implemention of CMA-ES without using numpy.
    • cma.recombination_weights - No summary
    • cma.restricted_gaussian_sampler - VD-CMA and VkD-CMA
    • cma.s - versatile shortcuts for quick typing in an (i)python shell or even from cma.s import * in interactive sessions.
    • cma.sampler - Collection of classes that sample from parametrized distributions and provide an update mechanism of the distribution parameters.
    • cma.sigma_adaptation - step-size adaptation classes, currently tightly linked to CMA, because hsig is computed in the base class
    • cma.test - test module of cma package.
    • cma.transformations - Search space transformation and encoding/decoding classes
    • cma.utilities - various unspecific utilities
      • cma.utilities.math - various math utilities, notably eig and a collection of simple functions in Mh
      • cma.utilities.python3for2 - to execute Python 3 code in Python 2.
      • cma.utilities.utils - various utilities not related to optimization
    • cma.wrapper - Interface wrappers for the cma module.
