cma.transformations.DiagonalDecoding

class documentation

class DiagonalDecoding(AdaptiveDecoding):

View In Hierarchy

Diagonal linear transformation.

Supports self * a, a * self, a / self, self *= a, as if self is an np.array. Problem: np.array does broadcasting.

>>> import cma
>>> from cma.transformations import DiagonalDecoding as DD

References: N. Hansen (2008). Adaptive Encoding: How to render search coordinate system invariant. In PPSN Parallel Problem Solving from Nature X, pp. 205-214.

sm = StatisticalModel...()
ad = AdaptiveDecoding...()

y = ad * z  # decoding applied
x = m + y  # candidate solution
ad.tell([sm.transform_inverse(z)], [0.1])
sm.update([y / ad], [0.01]) # remark that y / ad != z

where the symmetric transformation sm.transform_inverse(z) makes z isotropic.
