cma.purecma.DecomposingMatrix(SquareMatrix) class documentation

Part of cma.purecma (View In Hierarchy)

Symmetric matrix maintaining its own eigendecomposition.

If isinstance(C, DecomposingMatrix), the eigendecomposion is the return values of eig and stored in the attributes eigenbasis and eigenvalues such that the i-th eigenvector is:

[row[i] for row in C.eigenbasis]  # or equivalently
[C.eigenbasis[j][i] for j in range(len(C.eigenbasis))]

with eigenvalue C.eigenvalues[i] and hence:

C = C.eigenbasis x diag(C.eigenvalues) x C.eigenbasis^T
